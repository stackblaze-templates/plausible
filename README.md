# Plausible

Simple and privacy-friendly analytics. No cookies, fully GDPR/CCPA compliant.

## Local Development

```bash
docker compose up
```

Visit `http://localhost:8000` to set up Plausible.

Note: Plausible requires both PostgreSQL and ClickHouse.

## Deploy on StackBlaze

This template includes a `stackblaze.yaml` that provisions Plausible with PostgreSQL.
